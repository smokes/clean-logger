## Installation 
```bash
$ npm install clean-logger
```
## How to use?
```js
const log = require("clean-logger");

log("a simple log messge");

log.error("a simple error message"); 
```
## API Functions

- log(message);
- log.error(message);
- log.debug(message);
- log.warn(message);
- log.info(message);
